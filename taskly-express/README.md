## Ejecucion de desarrollo

Instalacion de dependencias

```bash
npm i
```

Crear el servidor

```bash
npm run dev
```

## Ejecucion con Docker

Creación del contenedor
```bash
docker build -t <name> .
```

Ejecución del contenedor
```bash
docker run -p 4200:80 <name_container>
```
